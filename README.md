# Lewis Store Welcome Page

A simple interactive webpage built with HTML, CSS, and JavaScript for the Lewis Furniture Store.

## 🖥️ Demo

Open `index.html` in your browser to try it out.

## ✨ Features

- Displays a welcome message for the store
- **View Customer Message** button updates the page text and shows a popup alert
- **Change Page Theme** button toggles between light and dark mode
