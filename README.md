# Lewis Store Welcome Page

A simple interactive webpage built with HTML, CSS, and JavaScript for the Lewis Furniture Store.

## 🖥️ Demo

Open `index.html` in your browser to try it out.

## ✨ Features

- Displays a welcome message for the store
- **View Customer Message** button updates the page text and shows a popup alert
- **Change Page Theme** button toggles between light and dark mode

## 📁 Project Structure

lewis-store-welcome/
├── index.html
├── style.css
└── script.js


## 🚀 Getting Started

1. Clone the repository
```bash
   git clone https://github.com/your-username/lewis-store-welcome.git
```
2. Open `index.html` in your browser

No build tools or dependencies required — pure HTML, CSS, and JavaScript.

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (DOM manipulation, event listeners)

## 📚 Concepts Practiced

- Selecting elements with `getElementById`
- Handling click events with `addEventListener`
- Updating page content with `textContent`
- Toggling CSS classes with `classList.toggle`
